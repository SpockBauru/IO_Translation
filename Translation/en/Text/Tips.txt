//============================== Tips ==============================

Ctrl+Alt+テンキー4～5 同時押しで画面をそのままカメラに設定できます=Pressing Ctrl + Alt + keypad 4, 5 or 6 at the same time saves a camera preset. Press keypad 4/5/6 to load saved camera presets.
[Tキー]移動    [Rキー]回転=[T key] Move [R key] Rotate
「@」､「:」　カメラを左右に回転、同時押しでリセット=The " and : keys rotate the camera, press them simultaneously to reset rotation.
エッチシーンでプラスキーを長押しでランダムでスピードが変化するモードになります=Press and hold the plus key during the sex scene to enter a mode where the speed changes at random.
カスタムデータは「CustomData」フォルダ内にあります。\n4つ以上残したいときは予備としてフォルダ外に保存することをお勧めします。=Your custom character presets are located in the "Save/CustomData" folder.\nIf you want to keep more than four presets, it is recommended to save them outside the folder as a backup.
カスタム中、「←」、「→」でキャラを回転、「↑」で停止、「↓」でリセット=During customization, use "←" and "→" to rotate the character, "↑" to stop, and "↓" to reset.
フルキー「1」　男の表示が切り替わります=Number key "1" changes the visibility of male characters.
フルキー「2」　男性器の表示が切り替わります=Number key "2" changes the visibility of male genitals.
フルキー「3」　主人公の服装の表示/非表示=Number key "3" changes the visibility of the main character's clothes.
フルキー「4」　背景の表示が切り替わります=Number key "4" changes the visibility of the background.
フルキー「5」　カメラターゲットの表示/非表示=Number key "5" changes the visibility of the camera target (center point when moving the camera).
フルキー「6」　遮断物の半透明化=Number key "6" changes the visibility of obstructions (objects that can get in the way of the camera).
フルキー「7」　キーパッドの表示/非表示=Number key "7" changes the visibility of the on-screen keypad.
フルキー「8」　断面図の表示/非表示=Number key "8" changes the visibility of the X-Ray window.
動作が重いときは環境設定で「Performance」にし、影とエフェクト系をoff、鏡を非表示にするとかなり軽くなります=If the game isn't running smoothly, set the quality setting to "Performance" and disable shadows, effects, and the mirror.
環境設定の「マニュアル」からオンラインマニュアルが開けます。\n操作が分からないときは活用してみてください=You can open the online manual from "Manual" in the Settings. \nPlease make use of it if you have any questions.
環境設定の「画面設定2」でカスタム状態をADVに反映できます=Custom character presets can be enabled in ADV scenes, look in the "Graphic Settings 2" tab of Settings.
環境設定の「画面設定2」で尿、母乳の設定ができます=You can set urine and breast milk in the "Graphic Settings 2" tab of Settings.
Ｆ１キーで環境設定を開く事ができます=You can open settings with the F1 key.
